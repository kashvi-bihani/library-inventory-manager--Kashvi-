#  Library Management System

A command-line library catalog system built using object-oriented programming, JSON persistence, and modular packaging.


##  Features
 Add new books to the library  
Search books by title or ISBN  
 Issue & return books with status tracking  
 Display complete inventory  
 Automatic saving/loading using JSON  
 Modular package structure for maintainability
